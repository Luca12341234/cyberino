🔍 60 motori di ricerca e database OSINT per la cybersecurity (2025)

Open‑Source Intelligence (OSINT) significa raccogliere informazioni da fonti aperte — senza dover “bussare” a sistemi altrui. Dai motori che scansionano l’intero IPv4, ai database di vulnerabilità, fino ai cataloghi di leak: ecco 60 strumenti indispensabili (e gratuiti, almeno in parte) per analisi di sicurezza, threat hunting e red teaming.
Ultimo aggiornamento: luglio 2025.

1. Motori di ricerca generici

  • Google (google.com) — Il classico: indispensabile per Google Dorks e ricerche mirate con operatori avanzati.
  • Bing (bing.com) — Meno affollato, con indici leggermente diversi da Google e API interessanti per l’automazione.

2. Scansione di Internet e dispositivi IoT

  • Shodan (shodan.io) — Il “motore di ricerca per dispositivi connessi”: banner, servizi e CVE correlati.
  • Censys (censys.io) — Dataset IPv4/IPv6 e certificati X.509, con query SQL‑like.
  • ZoomEye (zoomeye.org) — Alternativa cinese a Shodan; ottimo per asset discovery in Asia‑Pacifico.
  • FOFA (fofa.info) — Indice gigantesco (> 30 miliardi di host) e filtri su titolo HTML, header, favicon hash.
  • Onyphe (onyphe.io) — Combina scansioni mass‑scan + intelligence da leak e malware C2.
  • Netlas (netlas.io) — Query JSON flessibili su TLS, WHOIS, DNS passivo, favicon, favicon hash.
  • Recon.dev (recon.dev) — API specializzata in scansioni Certificate Transparency e sottodomini.
  • Chaos Project (chaos.projectdiscovery.io) — Dataset gratuito di sub‑domain enumeration, pensato per subfinder.

3. Rumore, traffico e contesto

  • GreyNoise (greynoise.io) — “Spam filter” per Internet: mostra se un IP genera rumore di scansioni note.

4. Vulnerabilità, exploit e advisory

  • NIST NVD (nvd.nist.gov) — Database CVE ufficiale: severity, CPE e metriche CVSS.
  • osv.dev (osv.dev) — Advisory machine‑readable per ecosistemi open source (Go, npm, PyPI…).
  • VulnIQ (vulnIQ.com) — Analisi CVE arricchite con PoC e correlazioni cronologiche.
  • VulDB (vuldb.com) — Community con timeline di exploit‑in‑the‑wild.
  • 0day.today (0day.today) — Marketplace/archivio di exploit proof‑of‑concept.
  • Rapid7 DB (rapid7.com/db) — Ricerca CVE collegata ai moduli Metasploit.
  • Exploit‑DB (exploit-db.com) — Script exploit, fuzzing e cheatsheet off‑the‑shelf.
  • Vulmon (vulmon.com) — Aggregatore CVE con changelog e feed RSS personalizzati.

5. Certificati, TLS e enumerazione sub‑domain

  • crt.sh (crt.sh) — Ricerca Certificate Transparency log, essenziale per scoprire host nascosti.
  • C99.nl (c99.nl) — Interfaccia semplificata per query CT log con filtri wildcard.
  • FullHunt (fullhunt.io) — Monitoraggio continuo di certificati, subdomini, takeover.
  • URLScan.io (urlscan.io) — Screenshot, DOM e richieste di qualunque URL scansionato (utile per phishing).

6. DNS & co.

  • DNSdb (dnsdb.io) — DNS passivo storico (tbytes di record A/CNAME/TXT…).
  • DNSViz (dnsviz.net) — Visualizza delegazioni, DS, RRSIG: perfetto per troubleshooting DNSSEC.
  • RobTex (robtex.com) — Mappe di “vicinanza” DNS/IP e correlazioni MX/ASN.

7. Ricerca nei repository di codice

  • SearchCode (searchcode.com) — Regex su 75+ milioni di progetti OSS.
  • grep.app (grep.app) — Ricerca immediata su GitHub pubblici; supporta look‑behind/PCRE.
  • NerdyData (nerdydata.com) — Trova frammenti di codice riutilizzati fra domini.

8. OSINT per domini, tecnologia e business

  • BuiltWith (builtwith.com) — Fingerprinting di stack web (CMS, CDN, librerie JS…).
  • Hunter.io (hunter.io) — Enumerazione e‑mail con verifica MX/SMTP.
  • SpyDialer (spydialer.com) — Lookup di numeri telefonici e reverse phone.
  • ThatsThem (thatsthem.com) — People search: email ↔ indirizzo ↔ telefono.
  • Snusbase (snusbase.com) — Query su db di leak combinati (email, username, hash).
  • Dehashed (dehashed.com) — Motore di breach, include ricerche per password hash e dati geografici.
  • LeakIX (leakix.net) — Scansiona bucket S3, endpoint Elasticsearch, Git repo pubblici.

9. Living‑off‑the‑Land & misconfig

  • GTFOBins (gtfo.mitre.org) — Bypass privilege escalation con binari “benigni” di Linux.
  • LOLBAS (lolbas-project.github.io) — Controparti Windows di GTFOBins.

10. Wireless & geolocalizzazione

  • WiGLE (wigle.net) — Milioni di reti Wi‑Fi wardriving, visibili su mappa.
  • WiFiMap (wifimap.io) — Hotspot pubblici con velocità stimata.
  • WiFiSpc (wifispc.com) — Database crowdsourced di SSID e coordinate.

11. Telecamere ed IoT esposti

  • Insecam (insecam.org) — Live feed di videocamere IP senza password o con credenziali di default.

12. Dark web & Tor

  • Ahmia (ahmia.fi) — Indicizza hidden‑services onion “puliti” (no abusi).
  • tor.link (tor.link) — Ricerca onion + bridge, filtro per lingua.
  • ORKL (orkl.org) — Directory open‑source di servizi Tor con ranking reputazionale.

13. File, malware & sandbox

  • FileSec (filesec.io) — Ricerca in drop‑zone pubbliche (FTP, SMB, WebDAV).
  • MalAPI (malapi.io) — “Cheat sheet” interattivo di funzioni Win32 usate dal malware.
  • Tria.ge (tria.ge) — Sandbox cloud automatizzata per PE, APK, DOCM, ecc.

14. Miscellanea

  • Riddler (riddler.io) — Cerca stringhe nei banner di servizio (es. “Welcome to Apache”).
  • Omnisint (omnisint.io) — Bulk subdomain enumeration + wordlist generate.
  • SynapsInt (synapsint.com) — Aggregatore OSINT multilingua: domini, WHOIS, leak.
  • TinEye (tineye.com) — Reverse‑image search alternativo ai servizi BigTech.
  • mylnikov.org (mylnikov.org) — Database open di geo‑IP e proxy/VPN noti.
